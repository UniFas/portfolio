Fear Index Overview

The Fear Index (0–100) dynamically represents perceived threat. It grows based on light exposure, noise, movement intensity, and gaze-based attention.

Player Action Values

Action Fear + Duration Notes
Walk 0 - Safe state
Run +2/sec Active Tension ramps gradually
Use Light +1/sec Active Attracts monster
Stay in Dark +3/5 sec Passive Fear of unknown
Look at Monster +15 3 sec High intensity
Make Noise +20 Instant Trigger moment

Scenario Example

Player runs → Fear reaches 60 → Monster blocks corridor → Player hides → Fear drops → Monster retreats. This creates peaks followed by controlled relief.
